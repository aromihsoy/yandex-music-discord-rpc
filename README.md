# Yandex Music Discord RPC 🎵

Легковесное приложение для Windows, которое транслирует текущий трек из Яндекс Музыки в статус Discord (Rich Presence). Программа работает в фоновом режиме и управляется через системный трей.

## Особенности
* Трансляция названия трека, имени исполнителя и статуса (играет/пауза) в Discord.
* Автоматическая подгрузка обложек альбомов через Yandex Music API.
* Работа в фоновом режиме с иконкой в системном трее Windows.
* Безопасное хранение ID приложения через переменные окружения.
* Автономная работа скомпилированного `.exe` файла без необходимости открывать консоль.

## Стек технологий
* **Python 3**
* `pypresence` — интеграция с Discord API.
* `pystray` & `Pillow` — создание интерфейса в системном трее.
* `python-dotenv` — управление переменными окружения.
* `yandex-music` — получение метаданных и обложек треков.
* `winrt` — взаимодействие с Windows Media Transport Controls.

## Установка и запуск (Для пользователей)
Самый простой способ начать использовать программу — скачать готовый релиз.
1. Перейдите в раздел **Releases** на этой странице.
2. Скачайте архив последней версии.
3. Распакуйте архив и запустите `YandexMusicRPC.exe`.
4. Программа появится в системном трее (возле часов). Для закрытия нажмите по иконке правой кнопкой мыши и выберите «Выход».
